# Wholesale Customer Segmentation using K-Means

##  Project Title
Wholesale Customer Segmentation using K-Means Clustering

##  Objective
The main objective of this project is to segment wholesale customers into different groups based on their purchasing behavior using the K-Means clustering algorithm.

## Dataset
The dataset used in this project contains information about customers' annual spending on various product categories such as:
- Fresh products
- Milk
- Grocery
- Frozen products
- Detergents_Paper
- Delicatessen

##  Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

##  Algorithm Used
- K-Means Clustering

##  Process
1. Data loading and preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature scaling using StandardScaler  
4. Applying K-Means clustering  
5. Determining optimal clusters using Elbow Method  
6. Visualizing clusters  

##  Output
The customers were successfully segmented into different clusters based on their purchasing patterns. Each cluster represents a group of customers with similar behavior.

##  Conclusion
This project helps businesses understand customer groups and make better marketing strategies, improving sales and customer satisfaction.
